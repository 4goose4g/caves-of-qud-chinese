# 卡德洞窟(caves of qud) 汉化
游戏内UI界面的部分字符是硬编码在脚本内的，没有源码只能反编译，本人反编译小白，反编译后重新生成一直失败

目前只汉化caves of qud的xml文件内英文描述，大概看了下xml文件，覆盖面还很广

**新进展**：使用Mono.Cecil修改dll内的字符文本然后重新打包成功，这样可以对硬编码部分的字符内容进行汉化
